## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/rafaelshembek/first_project.git
```

Entre no diretório do projeto

```bash
  cd first_project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npx expo start
```


## Stack utilizada

**Front-end:** React Native
